# Grant's 100

A 100-day strength tracker built around a 4-day **Pull / Push / Lower / Upper** split, four workouts per week.

- Logs weight and reps for every set
- Shows last session's numbers so you know what to beat — by reps, by weight, or both
- Built-in rest timer, weekly recap, and a progress chart for every lift
- Saves on your device, no account or server needed

A single self-contained `index.html`. Open it in any browser, or add it to your phone's home screen to run it full-screen like an app.

## Deploy
Hosted on GitHub Pages from the `main` branch.
